#Introdução

Este relatório apresenta o processo de implementação de um código em “C” para Arduino para controle de temperatura envolvendo um sensor de temperatura um ventilador e uma alarme via buzzer.

#Metodologia do Trabalho

O código foi implementado via simulador thikercad  mas pode ser executado em outros softwares de simulação.

#Premissas de Funcionamento

Fazer a leitura da temperatura;
Fazer o acionamento de um motor de ventilador caso a temperatura seja igual ou maior a 30 °C; 
Caso a temperatura ultrapasse os 50 °C, um LED vermelho e uma buzina devem acionar avisando uma situação de emergência.
